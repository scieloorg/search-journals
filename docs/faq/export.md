# Exportar Resultados

Aprenda como salvar e exportar seus artigos em diferentes formatos.

## Formatos de Exportação Disponíveis

### RIS (Reference Information System)

**O que é:** Formato padrão para gerenciadores de referências bibliográficas

**Para usar:** 
- Zotero
- Mendeley
- EndNote
- RefWorks

**Exemplo:** Importa automaticamente título, autor, ano, revista, etc.

**Como usar:**
```
1. Selecione artigos
2. Clique em "Exportar"
3. Escolha "RIS"
4. Abra em seu gerenciador de referências
5. Pronto! Citações organizadas
```

---

### BibTeX

**O que é:** Formato para citações em LaTeX e documentos científicos

**Para usar:**
- LaTeX
- Overleaf
- Documentos acadêmicos

**Exemplo de saída:**
```
@article{Silva2023,
  author = {Silva, J.},
  title = {Título do Artigo},
  journal = {Revista Científica},
  year = {2023},
  volume = {10},
  pages = {123-145}
}
```

**Como usar:**
```
1. Exporte em BibTeX
2. Cole no seu arquivo .bib
3. Cite no seu documento com \cite{Silva2023}
```

---

### CSV (Comma-Separated Values)

**O que é:** Formato de planilha - abra no Excel, Google Sheets, etc.

**Para usar:**
- Microsoft Excel
- Google Sheets
- LibreOffice Calc
- Qualquer editor de planilhas

**Exemplo de saída:**
```
Título,Autor,Ano,Revista,DOI,URL
"Artigo 1",Silva J.,2023,Journal X,10.xxx,http://...
"Artigo 2",Santos M.,2023,Journal Y,10.xxx,http://...
```

**Como usar:**
```
1. Exporte em CSV
2. Abra no Excel ou Google Sheets
3. Organize, filtre, analise conforme necessário
4. Exporte novamente se precisar em outro formato
```

---

### Citação (Formato Texto)

**O que é:** Citações prontas em formato padronizado

**Formatos inclusos:**
- ABNT (padrão brasileiro)
- APA (padrão internacional)
- Vancouver (medicina)
- Chicago
- Harvard

**Como usar:**
```
1. Selecione os artigos
2. Clique em "Exportar"
3. Escolha "Citação"
4. Selecione o formato desejado
5. Copie as citações para seu trabalho
```

**Exemplo ABNT:**
```
SILVA, J. et al. Título do artigo. Revista Científica, 
São Paulo, v. 10, n. 2, p. 123-145, 2023.
```

## Opções de Exportação

### Quais artigos exportar?

A plataforma oferece 3 opções:

#### 1️⃣ Página Atual

Exporta apenas os artigos que estão sendo visualizados na página

```
Quando usar: Poucos artigos relevantes
Quantidade: Tipicamente 10-50 artigos
```

#### 2️⃣ Todos os Resultados

Exporta TODOS os artigos encontrados (até o limite da plataforma)

```
Quando usar: Revisão sistemática ou análise completa
Quantidade: Pode ser centenas ou milhares
Limitação: Algumas plataformas limitam a 1.000-5.000
```

#### 3️⃣ Seleção

Exporta apenas os artigos que você marcou manualmente

```
Quando usar: Curadoria manual dos melhores
Quantidade: Conforme você selecionou
Recomendado: Melhor qualidade, mais trabalho
```

## Passo a Passo: Exportar para o Mendeley

### Objetivo: Organizar suas referências bibliográficas

1. **Realize sua busca**
   ```
   Digite: educação AND tecnologia
   Aplique filtros desejados
   Resultado: 500 artigos
   ```

2. **Selecione os artigos**
   ```
   Opção 1: Marque individualmente (☑ Artigo 1, ☑ Artigo 2...)
   Opção 2: Selecione "Todos os resultados"
   ```

3. **Exporte em RIS**
   ```
   Clique em: "Exportar"
   Selecione: "RIS"
   Arquivo baixado: export.ris
   ```

4. **Abra o Mendeley**
   ```
   Menu: File → Import → Bibtex/RIS
   Selecione: export.ris
   ```

5. **Organize no Mendeley**
   ```
   Crie pastas por tema
   Adicione notas e destaques
   Suas referências estão prontas para citar!
   ```

---

## Passo a Passo: Exportar para Excel

### Objetivo: Analisar dados em planilha

1. **Realize sua busca**
   ```
   Digite: artigos sobre saúde
   Ordene por: Mais recentes
   ```

2. **Exporte em CSV**
   ```
   Selecione: Todos os resultados
   Clique em: "Exportar"
   Formato: CSV
   Arquivo: resultados.csv
   ```

3. **Abra no Excel**
   ```
   Arquivo → Abrir
   Selecione: resultados.csv
   ```

4. **Organize e analise**
   ```
   Coluna A: Título
   Coluna B: Autor
   Coluna C: Ano
   Coluna D: Revista
   
   Agora você pode:
   - Filtrar por ano
   - Contar artigos por revista
   - Buscar autores específicos
   - Criar gráficos
   ```

## Formatação de Citações

### Qual formato escolher?

| Uso | Formato Recomendado |
|-----|---|
| Trabalhos acadêmicos no Brasil | ABNT |
| Artigos científicos internacionais | APA |
| Revistas de medicina | Vancouver |
| Humanidades | Chicago |
| Ciências da computação | IEEE |

### Como saber qual usar?

Verifique:
- **Orientações da universidade** (projeto, dissertação, tese)
- **Normas da revista** (se for submeter artigo)
- **Padrão da sua área** (converse com seu orientador)

## Dicas para Exportação

### ✅ Melhores práticas:

- **Exporte em RIS** para máxima compatibilidade com gerenciadores
- **Use CSV** quando precisar analisar dados em planilha
- **Revise a exportação** - garanta que todos os campos vieram corretos
- **Faça backup** das referências exportadas
- **Organize por pastas** - temas, projetos, disciplinas

### ❌ Evite:

- Exportar **formatos antigos** - prefira RIS/BibTeX
- **Perder o arquivo original** - sempre tenha backup
- Exportar **sem seleção criterosa** - muitos artigos desnecessários
- **Misturar formatos** - use um padrão consistente

## Compartilhando Resultados

### Formas de compartilhar:

**1. Enviar lista por e-mail**
```
Exporte em CSV ou formato legível
Anexe o arquivo à mensagem
Colegas abrem em Excel/Sheets
```

**2. Compartilhar URL da busca**
```
Copie a URL da sua busca
Envie para colega
Ele pode refazer a mesma busca
```

**3. Exportar e compartilhar no Mendeley**
```
Crie biblioteca colaborativa
Compartilhe acesso com colegas
Todos veem e organizam juntos
```

## Limitações e Considerações

### Limite de Registros

Algumas plataformas limitam quantos registros podem ser exportados por vez:
- Até 1.000-5.000 registros é comum
- Se precisar mais, faça várias exportações filtradas

### Tamanho do Arquivo

Arquivos grandes (>10MB) podem demorar para baixar:
- Use formatos comprimidos (ZIP)
- Exporte em lotes menores se necessário

### Informações Disponíveis

Nem todos os campos estão em todos os formatos:
- RIS: Tem mais campos
- CSV: Mais básico
- Citação: Apenas texto formatado
