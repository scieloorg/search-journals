# Filtros e Organização

Refine seus resultados usando filtros disponíveis na plataforma.

## Filtros Disponíveis

A plataforma oferece vários filtros para refinar a busca:

### Por Período

```
Filtre resultados por ano(s) de publicação
Exemplo: Mostrar apenas artigos de 2022-2023
```

### Por Periódico/Revista

```
Busque artigos apenas de revistas específicas
Exemplo: Nature, Lancet, Science, etc.
```

### Por Idioma

```
Escolha o idioma dos artigos
Opções: Português, Inglês, Espanhol, Francês, etc.
```

### Por Tipo de Documento

```
Filtre por tipo de publicação
Exemplo: Artigos originais, Revisões, Editoriais, etc.
```

### Por Área Temática

```
Especialize em áreas científicas
Exemplo: Medicina, Tecnologia, Educação, etc.
```

---

## Como Aplicar Filtros

### Passo a passo:

1. **Realize uma busca inicial**
   ```
   Digite: educação
   Clique: Buscar
   ```

2. **Localize os filtros** (geralmente à esquerda ou acima dos resultados)

3. **Selecione os filtros desejados**
   ```
   ☐ Ano: 2023
   ☐ Revista: Revista de Educação
   ☐ Idioma: Português
   ☐ Tipo: Artigos Originais
   ```

4. **Aplique os filtros**
   - Os resultados se atualizam automaticamente
   - Ou clique em "Aplicar filtros"

## Ordenação de Resultados

A plataforma oferece formas de organizar seus resultados:

### Opções comuns de ordenação:

| Opção | O que mostra primeiro |
|-------|---|
| **Relevância** | Artigos mais relacionados à sua busca |
| **Mais recentes** | Publicações mais novas (ano descendente) |
| **Mais antigos** | Publicações mais antigas (ano ascendente) |
| **Mais citados** | Artigos com mais citações em outras obras |
| **Mais acessados** | Artigos mais lidos pelos usuários |

### Quando usar cada opção:

| Situação | Use |
|----------|-----|
| Você quer os melhores artigos | Relevância |
| Você quer pesquisa atualizada | Mais recentes |
| Você estuda história da ciência | Mais antigos |
| Você quer artigos influentes | Mais citados |
| Você quer tendências atuais | Mais acessados |

## Número de Resultados por Página

Escolha quantos artigos ver por página:

### Opções típicas:

- **10 resultados** - Visão rápida, menos scroll
- **25 resultados** - Bom balanço entre quantidade e performance
- **50 resultados** - Para análise profunda
- **100 resultados** - Para pesquisa intensiva

**Dica**: Em conexões lentas, use menos resultados por página.

## Clusters e Facetas

Muitas plataformas mostram agrupamentos automáticos dos resultados.

### O que são clusters?

São **agrupamentos temáticos** dos resultados encontrados.

### Exemplo:

Busca: `tecnologia`

```
Clusters encontrados:
├─ Educação e Tecnologia (1.234 artigos)
├─ Inteligência Artificial (892 artigos)
├─ Tecnologia e Saúde (756 artigos)
├─ Computação (634 artigos)
└─ Tecnologia Ambiental (412 artigos)
```

### Como usar:

Clique em um cluster para refinar resultados para esse subtema!

## Dicas de Ouro

### ✅ Melhores práticas:

- Comece com **filtros abrangentes**, depois refine
- Use **combinações de filtros** para resultados precisos
- **Salve suas buscas** para reutilizar depois
- **Experimente ordenações** diferentes para novos insights

### ❌ Evite:

- Aplicar **muitos filtros de uma vez** (pode zerar resultados)
- Filtrar por **ano muito antigo** (literatura desatualizada)
- **Esquecer de desmarcar filtros** quando fizer nova busca
