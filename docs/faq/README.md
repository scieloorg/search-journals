## 🔍 Começar Rápido

### Perguntas mais frequentes:

**P: Como faço uma busca simples?**  
R: Digite o termo que procura na caixa de busca e clique em "Buscar". [Leia mais...](./search-basics.md#como-fazer-uma-busca-simples)

**P: Posso buscar uma expressão exata?**  
R: Sim! Use aspas. Por exemplo: `"educação física"`. [Leia mais...](./search-operators.md#busca-exata-com-aspas)

**P: O que é a truncagem com asterisco (*)?**  
R: É uma técnica para buscar todas as variações de uma palavra. Por exemplo: `pesquis*` encontra pesquisa, pesquisador, pesquisas, etc. [Leia mais...](./search-operators.md#truncagem-com-asterisco)

**P: Como combino vários termos na mesma busca?**  
R: Use os operadores AND, OR ou NOT. Por exemplo: `educação AND tecnologia`. [Leia mais...](./search-operators.md#operadores-booleanos)

**P: Qual é o limite de caracteres para uma busca?**  
R: A plataforma suporta até 1024 combinações de termos, o que é mais do que suficiente para buscas normais. [Leia mais...](./technical.md#limite-de-caracteres)

## 📚 Dicas Gerais

- **Seja específico:** Use termos precisos relacionados ao seu tema
- **Combine filtros:** Use autor, ano, revista, etc. para refinar resultados
- **Experimente sinônimos:** Se não encontrar resultados, tente palavras alternativas
- **Use aspas para termos compostos:** `"análise de variância"` em vez de análise de variância
- **Aproveite a truncagem:** `climat*` é mais amplo que `clima`
